# Shadcn Dashboard

A modern, responsive dashboard built with Next.js and shadcn/ui, featuring interactive data visualizations powered by Nivo charts.

## Features

* Interactive charts: bump, calendar heatmap, line charts, and data tables
* Clean, modern UI using shadcn/ui and Tailwind CSS
* Responsive dashboard layout

## Tech Stack

* Next.js 16.0.6
* React 19.2.0
* TypeScript
* Tailwind CSS 4
* shadcn/ui
* Nivo Charts
* lucide-react

## Getting Started

```bash
git clone https://github.com/AnkitShaw-100/Shadcn.git
cd Shadcn/frontend
npm install
npm run dev
```

Visit `http://localhost:3000`.

## Build & Deploy

```bash
npm run build
npm start
```

Supports easy deployment on Vercel.

## Project Structure

```
frontend/
├── app/            # App router and pages
├── components/     # Dashboard and UI components
├── lib/            # Utilities
```
